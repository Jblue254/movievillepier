# Movieville 

## About the Project

Movieville is a React-based movie watch list application that allows users to manage a personal collection of movies. Users can add new movies to their watch list, view them in a clean and responsive interface, and remove movies they no longer want to track.

## Features

* Add new movies to the watch list
* Remove movies from the list
* Dynamic rendering using React's `.map()` function
* Automatic year assignment when adding movies
* Component-based architecture
* Responsive design with Tailwind CSS
* Clean and intuitive user interface

## Getting Started

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/movieville.git
```

Navigate to the project directory:

```bash
cd movieville
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Visit the deployed application:

**Live Site:** https://your-live-demo-link.vercel.app/

---

## Usage

1. Enter the movie title.
2. Enter the movie genre.
3. Click **Add Movie**.
4. View the movie in your watch list.
5. Click **Remove** to delete a movie from the list.
